Introduction of Set Theory through Halmos' Naive Set Theory

I

Set Theory, broadly speaking, is the branch of mathematics that deals with the formal properties of sets. Sets, as they are usually conveived, have elements or members. A set itself may also be an element of some other set.

The principle concept of set theory is that of belonging. If x belongs to A, or if x is an element or member of A, then we shall write that:

$x \epsilon A$

Belonging is not always reflexive. Belonging is not transitive as well.

Another principle concept of set theory is that of equality, which is just:

$A=B$

The most basic axiom of set theory, therefore, is the relationship between the property of belonging and the property of equality, which is the Axiom of Extension:

Axiom of Extension: Two sets are equal if and only if they have the same elements

Moving on to subsets, if A and B are sets and if every element of A is an element of B, then we can say that A is a subset of B, and we can write:

$A \subset B$

The definition implies that each set must be included in itself, making set inclusion reflexive. Set inclusion can also be seen as transitive. Set inclusion is also antisymmetric, meaning that if A is a subset of B, B does not have to be a subset of A. Because of this property, almost all proofs of equalities between two sets A nad B are split into two parts, first show that A is a subset of B and then show that B is a subset of A.

If A is a proper subset of B, meaning that the two sets are not equal, then we can write:

$A \subseteq B$

Before moving onto the second axiom of set theory, lets pause for a momnet to explore set sentences. There are two basic types of sentneces, which is the assertions of belonging and the assertions of equality, with all other sentences obtained from these atomic sentneces by using other logical operatores.

The second axiom of set theory, the Axiom of Specification, states that:

The Axiom of Specification: To every set A and every condition S(x), there corresponds a set B whose elements are exactly those elements of x of A for which S(x) holds

In other words, given a set N, we can construct a subst x such that:

${[x| x \epsilon N \cap xS(x)]}$

A cool application of this axiom is to prove that there is no universe. Consider the condition that x is not a member of x. Then it logically follows that if B = {x| x is a member of A and x is not a member of x}, then y is a member of set B if and only if y is a member of A and y is not a member of y So therefore, can it be that B is a member of A? After caseworking this problem into 2 cases, we find out that if B is a member of B, then B is not a member of B, a contradiction. If B is not a member of B, then B must be a member of B, another contradiction. Therefore, we arrive at the conclusion that B is not a member of A, which means that nothing contains everything, or there is no such thing as a universe set. This is called the famous Russell Paradox

Are there enough sets to ensure that every set is an element of some set. Is there a principle that allows us to contruct new sets with old sets without any specification? The answer is the Axiom of Pairing:

Axiom of Pairing: For any two sets there exists a set that they both belong to

A pair is a set that contains only A and B and nothing else, abbreviated as {a, b}. A singleton is a set that only has another set as its element. This happens when we form a set with a and a, abbreviated as {a, a} or just {a}

The axiom of pairing ensures that every set is an element of some set and that ny two sets are simultaenously elements of some one and the same set

One way of describing a united set is to require it to contain all the elements that belong to at least one of the two members of pair {A, B}. This is described with the Axiom of Unions:

Axiom of Unions: For every collection of sets there exists a set that contains all the elements that belong to at least one set of the given collection

This can be expressed as:

$U = {x|x \epsilon X for some X in C}$

, where C is a collection (of sets). In words, this is "For every collection C there exists a set U such that if x belongs to X for some X in C, then x belongs to U"

The simplest symbol for representing U is:

$\cup _{X \epsilon C} X$

The above reads as "The union of the set X for some X in collection C "

Here are some simple facts and properties about unions:

  • $\cup _{X \epsilon \emptyset} = \emptyset$
  • $\cup _{X \epsilon [A]} = A$
  • $\cup _{X \epsilon [A, B]} = A \cup B$

Other properties of unions are commutativity, associativity, and idempotence, which just means that the union of A and A is A.

A way to egneralize pairs through unions is through the fact that the union of the singletons {a} and {b} is the pair {a, b}. This fact could be extended to triplets and beyond.

If A and B are sets, the intersection of A and B is denoted and defined as:

$A \cap B = [x|x \epsilon A \cap x \epsilon B]$

The basic properties of intersection are similar to the basic properites of unions.

If the intersection of A and B is an empty set, then the sets A and B are called disjoint, hich basically means that they have no elements in common

To very useful facts about unions and intersections are the identities called the distributive laws:

$A \cap (B \cup C) = (A \cap B) \cup (A \cap C)$

$A \cup (B \cap C) = (A \cup B) \cap (A \cup C)$

Notice how the operations of union and intersection distribute into the bracketed elements, which retains the operation between them

If A and b are sets, the relative complement of B in A is the set defined by:

$A - B = [x|x \epsilon A \cap x \epsilon' B]$

Basically, the relative complement of B in A is just the elements that are in A but not in B, therefore, simplified into A - B. The absolute complement of A, denoted by A', is the relative complement of A in E, where E is the universe. From this, we get the following properties:

  • $(A')'=A$
  • $E'=\emptyset vice - versa$
  • $A \subset B if and only if B' \subset A'$

THe msot important statement regarding complements are the De Morgan Laws:

$(A \cup B)' = A' \cap B'$

$(A \cap B)' = A' \cup B'$

A good way to memorize this theorem is just to remember so that if you want to distribute the absolute complement into the brackets, make sure to flip the union and intersection operations.

If A and B are sets, then the Boolean Sum, or symmetric difference, of A and B is the set A + B defined by:

$A + B = (A - B) \cup (B - A)$

After considering the many properties of subsets, we now wonder whether or not sets themselves constitutes a set. The Axiom of Powers explains this:

Axiom of Power: For each set there exists a collection of sets that contains among its elements all the subsets of the given set

In other words:

$P = [X|X \subset C]$

The power set of a finite set with n elements has 2^n elements.

The ordered pairs of a and b, with first coordinate a and second coordinate b, is the set (a, b) defined by: (in terms of set theory)

$(a, b) = [[a], [a, b]]$

If A and B are sets, does there exist a set C that contains all the ordered pairs (a, b) with a in A and b in B? There does indeed exist a unique set A * B such that it consists exactly the ordered pairs (a, b) with a in A and b in B. This set is called the Cartesian Product of A and B:

A*B = [x|x = (a, b) for some a in A and for some b in B]

If the set X = [1, 2] and Y = [3, 4], then the cartesian product X * Y is [(1, 3), (1, 4), (2, 3), (2, 4)]
